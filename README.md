# Minecraft指令百科

## 项目说明
这是我闲暇时间写的一个Minecraft指令查询工具，主要方便自己和朋友查指令用。纯前端实现，打开就能用，不用联网（数据都在本地）。

## 功能特点

### 📱 移动端友好
- 响应式布局，手机平板都能正常看
- 侧边栏可以折叠，节省屏幕空间
- 表格支持横向滚动，小屏幕也能看全数据
- 按钮和链接都做了触摸优化，防止误触

### 🔍 搜索功能
- 支持模糊搜索，不用打完整指令
- 不管加不加"/"前缀都能搜到
- 实时过滤，输入即显示结果

### 📋 数据展示
- 卡片式布局，整洁好看
- 支持按版本和分类筛选
- 指令详情页有完整说明、语法和例子
- 分页显示，防止数据太多卡顿

### 🔄 版本支持
- 目前支持 1.8.0 ~ 1.21.11 版本
- 1.21 新增了重锤、旋风棒等物品数据
- 1.13 之前的版本用数字ID，之后用命名空间ID

## 技术栈
- **HTML5** + **CSS3** + **JavaScript (ES6+)**
- 纯前端实现，不需要服务器
- 没有用任何框架，原生JS写的

## 怎么用
直接双击 `index.html` 文件就能打开，不需要安装任何东西。

### 基本操作
1. **查指令**：在顶部搜索框输入关键词
2. **选版本**：左侧下拉菜单选择Minecraft版本
3. **看分类**：侧边栏点击分类可以筛选
4. **查详情**：点击指令卡片看详细说明
5. **移动端**：点击左上角按钮展开/折叠菜单
